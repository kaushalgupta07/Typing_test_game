# Typing Test Game

A simple and interactive typing speed test game built using **HTML**, **CSS**, and **JavaScript**.

## 🚀 Features

- Real-time typing test
- Timer and word-per-minute (WPM) calculation
- Accuracy measurement
- User-friendly interface
- Responsive design
